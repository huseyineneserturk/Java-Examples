## Getting Started

### Prerequisites

- Java Development Kit (JDK) 8 or higher
- An IDE (IntelliJ IDEA, Eclipse, or VS Code recommended)

### Running the Projects

1. Clone the repository:
```bash
git clone https://github.com/huseyineneserturk/Java-Examples.git
```

2. Navigate to the project directory:
```bash
cd Java-Examples
```

3. Open the desired project folder in your IDE

4. Run the main class for each project
